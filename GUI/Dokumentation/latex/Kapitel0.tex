\section{GUI starten}
\subsection{GUI aus Matlab starten}
Die GUI wird durch die Eingabe von "`\texttt{run \textcolor{pink}{MORLAB\_GUI}}"' im Command Window gestartet.
\subsection{GUI als executable starten}
Zum ausführen der GUI ohne Matlab ist es notwendig zunächst den MCRInstaller zu Installieren. Anschließend kann die .exe aus dem Ordner "`distrib"' geöffnet werden.
